**Steps to run:**

  **Clone repository:**
     
     git clone https://github.com/samansaeed2306/video-annotations.git
     
 **Install dependencies:**
 
     cd video-annotations
     npm i
     cd src
     npm i
     
   **Server:**
   
     cd video-annotations        
     npm run start: backend

  **Client:**
  
     cd video-annotations             
     npm run start: frontend
